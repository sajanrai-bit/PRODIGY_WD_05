# 🌦️ Weather App

A simple and responsive weather application built using **HTML, CSS, and JavaScript** that provides real-time weather updates using the **Open-Meteo API** without requiring an API key.

---

# ✨ Features

- 🌍 Automatic location detection
- 🔎 Search weather by city or place
- 🌡️ Temperature display in Celsius & Fahrenheit
- 💨 Wind speed information
- 💧 Humidity details
- ⏰ Real-time weather updates
- 📍 Reverse geolocation support
- 📱 Fully responsive UI
- 🔗 Share weather information
- ⚡ No API key required

---

# 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
- OpenStreetMap Nominatim API
- Responsive Web Design

---

# 🖥️ Application Features

## 🌍 Location Detection

Automatically detects the user's location using:

- Browser Geolocation
- IP-based fallback detection

---

## 🔎 Search Functionality

Users can search weather information for any city or place worldwide.

---

## 🌡️ Temperature Unit Toggle

Switch between:

- Celsius (°C)
- Fahrenheit (°F)

---

## 📊 Weather Information Display

Shows:

- Current temperature
- Weather condition
- Wind speed
- Humidity
- Time
- Coordinates

---

## 📤 Share Feature

Allows users to:

- Share weather details using device sharing
- Copy weather information to clipboard

# ⚙️ How to Run the Project

## Method 1: Run Directly

1. Download the project files
2. Open `weather.html` in any modern web browser

---

## Method 2: Using VS Code

1. Install Visual Studio Code
2. Install the **Live Server** extension
3. Open the project folder
4. Right-click `weather.html`
5. Click **Open with Live Server**
